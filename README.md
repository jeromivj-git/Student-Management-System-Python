# 🎓 Student Management System (Python)

A Python-based console application to manage student records efficiently.  
This system allows users to add, update, delete, search, and analyze student data, along with saving and loading records using Excel.

---

## 🚀 Features

- ✅ Add new student records  
- ✅ View all students  
- ✅ Search student by name  
- ✅ Update student details (name, age, marks)  
- ✅ Delete student records  
- ✅ Sort students by average marks  
- ✅ Filter students by performance (Excellent, Good, Average, Poor)  
- ✅ Automatic grade & performance calculation  
- ✅ Save data to Excel file  
- ✅ Load data from Excel file  

---

## 📊 Grading System

| Average Marks | Grade | Performance |
|--------------|------|------------|
| 90+          | A    | Excellent  |
| 75–89        | B    | Good       |
| 60–74        | C    | Average    |
| 40–59        | D    | Poor       |
| Below 40     | F    | Poor       |

---

## 🧠 Functional Highlights

- Validates user input (name, age, marks)  
- Automatically calculates:
  - Total marks  
  - Average  
  - Grade  
  - Performance  
- Updates grade and performance dynamically after mark changes  

---

## 🛠️ Technologies Used

- Python  
- openpyxl (for Excel file handling)
